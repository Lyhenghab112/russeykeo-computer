Basic HTML Structure

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Page Title</title> </head> <body> <h1>Main Heading</h1> <p>This is a paragraph.</p> </body> </html>

Creating Links

<a href="https://www.example.com">Visit Example.com</a> <a href="#section1">Go to Section 1</a> <a href="mailto:contact@example.com">Send Email</a>

Adding Images

<img src="image.jpg" alt="Description of image"> <figure> <img src="image.jpg" alt="Figure image"> <figcaption>Image caption</figcaption> </figure>
