Responsive Design

Media Queries

/* Mobile first approach */ .container { width: 100%; padding: 10px; } /* Tablet and up */ @media (min-width: 768px) { .container { width: 750px; margin: 0 auto; padding: 20px; } } /* Desktop and up */ @media (min-width: 1024px) { .container { width: 1000px; padding: 30px; } }

Responsive Container

This container adapts to different screen sizes. Try resizing your browser window!

Current behavior:

  • Mobile: Full width with 10px padding
  • Tablet: 750px width, centered with 20px padding
  • Desktop: 1000px width, centered with 30px padding

Responsive Units

.responsive-text { font-size: 1rem; /* Relative to root font size */ line-height: 1.5; } .responsive-width { width: 90%; /* Relative to parent */ max-width: 600px; /* Maximum width */ } .responsive-height { height: 50vh; /* 50% of viewport height */ }
