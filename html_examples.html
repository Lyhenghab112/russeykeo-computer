Practice Exercises

Exercise 1: Create a Personal Profile Page

Create an HTML page that includes:

  • Your name as the main heading
  • A profile picture
  • Personal information in a table
  • A list of your hobbies
  • A contact form

Exercise 2: Build a Recipe Page

Create an HTML page for a recipe that includes:

  • Recipe title and description
  • Ingredients list
  • Step-by-step instructions
  • Cooking time and difficulty
  • Nutritional information in a table

Exercise 3: Design a Product Catalog

Create an HTML page for a product catalog with:

  • Multiple product cards
  • Product images and descriptions
  • Pricing information
  • Add to cart buttons
  • Product categories
